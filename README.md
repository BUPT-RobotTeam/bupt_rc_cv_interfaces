>

## 环境要求
> 1. ROS2 (作者使用的平台是humble)

## msg关系图
```bash
┌───────────────────────┐       ┌───────────────────────┐
│                       │       │                       │
│     CVInferenceArray  │       │       CVCameras       │
│                       │       │                       │
│  - inference_result   │       │  - cam_type (uint8)   │
│    (CVInference[])    │       │  - cam_fps (float64)  │
│                       │       │                       │
│  - img (CVFrame)      │       └───────────────┬───────┘
│                       │                       │
└───────────────┬───────┘                       │
                |───────────────────────────────│
        ┌───────┴───────────────┐               │
        │                       │               │
        │       CVInference     │               │
        │                       │               │
        │  - track_id (int32)   │               │
        │  - tlwh (float32[])   │               │
        │  - score (float32)    │               │
        │                       │               │
        └───────────────────────┘               │
                                                │
                                  ┌─────────────┴─────────────┐
                                  │                           │
                                  │          CVFrame          │
                                  │                           │
                                  │  - frame_width (uint16)   │
                                  │  - frame_height (uint16)  │
                                  │  - frame_data (uint8[])   │
                                  │                           │
                                  └───────────────────────────┘
```
